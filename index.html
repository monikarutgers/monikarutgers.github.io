Summary: Latitude vs. X

Purpose of the project - Analyze Weather changes as you get closer to the Equator

Source and tools Used - OpenWeatherMap API, worked throug Pandas

Data set - 550 different cities.

The dataset was plotted using Matplotlib

We studied the following factors of weather vs.latitude:

  • Temperature
  • Cloudiness
  • Humidity
  • Wind Speed

This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of trends and correlations.
